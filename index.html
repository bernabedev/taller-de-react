¿Qué aprenderemos hoy?

  • Configuración de un proyecto React desde cero
  • Consumo de APIs externas (GitHub API)
  • Gestión de estados con useState y useEffect
  • Manejo de errores y estados de carga
  • Guardar usuarios favoritos (localStorage)
  • Componentes reutilizables

Nuestro Proyecto

Buscador de Usuarios GitHub

  • Buscar usuarios por nombre
  • Ver detalles de perfil
  • Añadir/eliminar favoritos
  • Gestionar respuestas y errores

Conceptos clave que dominaremos

  • Fetch API - Comunicación con servicios externos
  • async/await - Gestión de operaciones asíncronas
  • React Hooks - useState, useEffect
  • Arquitectura de componentes - Organización del código
  • LocalStorage - Persistencia de datos en el navegador

Posibles mejoras futuras

  • Modo oscuro / claro con Context API
  • Ver repositorios del usuario
  • Compartir perfiles por URL
  • Filtros de búsqueda avanzados
  • Estadísticas de actividad
  • PWA para uso offline

Requisitos para comenzar

  • Node.js - v16 o superior
  • Vite - Para la configuración rápida del proyecto
  • Editor de código - VS Code recomendado
  • Extensiones VS Code recomendadas:
    • ES7+ React/Redux/React-Native snippets
    • Prettier - Code formatter
    • ESLint
  • Git - Para control de versiones (opcional)

¿Listos para comenzar?

Aprenderemos React de la mejor manera: ¡construyendo algo real!

Comandos iniciales:

npm create vite@latest github-finder -- --template react
cd github-finder
npm install
npm run dev
